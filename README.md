# Portfolio – Uddhav Davey

A personal portfolio website built to showcase my projects, skills, experience, and resume.

## 🚀 Live Demo
[View Portfolio](https://uddhav-davey-portfolio.vercel.app/)

## 📌 About
This is a responsive and modern portfolio built using Next.js. It highlights my work in software development and Artificial Intelligence, along with selected projects and professional experience.

## 🛠️ Tech Stack
- Next.js
- React
- Tailwind CSS
- TypeScript

## ✨ Features
- Responsive UI for all devices
- Smooth navigation between sections
- Projects showcase section
- Experience and skills overview
- Downloadable resume
- Deployed on Vercel

## 📂 Project Structure
→ Reusable UI components
→ Next.js pages
→ Static assets (resume, images)
→ Global styles
